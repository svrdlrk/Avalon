# map-editor

Отдельный модуль редактора карт для Avalon.

## Как подключить в проект
В корневом `settings.gradle` должен быть include:

```gradle
include 'map-editor'
```

## Запуск
Из корня репозитория:

```bash
./gradlew :map-editor:run
```

Можно передать стартовые параметры:

```bash
./gradlew :map-editor:run --args="--project=/path/to/map.json --assets=/path/to/CastleWalls.zip"
```

## Уже работает
- окно Java 17 + JavaFX
- канвас с сеткой
- pan и zoom
- поиск по каталогу ассетов
- размещение объектов
- перемещение объектов
- удаление
- undo / redo
- сохранение и загрузка проекта в JSON
- импорт и экспорт layout DTO из `shared`

## Что удобно проверять вручную
1. открыть редактор;
2. выбрать ассет;
3. поставить несколько объектов;
4. переместить объект инструментом Move;
5. удалить объект;
6. нажать Undo / Redo;
7. сохранить проект;
8. закрыть и открыть снова;
9. импортировать / экспортировать layout JSON.
