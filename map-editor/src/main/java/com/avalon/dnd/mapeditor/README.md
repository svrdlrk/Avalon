# map-editor

Отдельный модуль редактора карт для Avalon.

## Как подключить в проект
В корневом `settings.gradle` должен быть include:

```gradle
include 'map-editor'
```

## Запуск
Из корня репозитория:

```bash
./gradlew :map-editor:run
```

Можно передать стартовые параметры:

```bash
./gradlew :map-editor:run --args="--project=/path/to/map.json --assets=/path/to/CastleWalls.zip"
```

## Уже работает
- окно Java 17 + JavaFX
- канвас с сеткой
- pan и zoom
- поиск по каталогу ассетов
- размещение объектов
- кисть для протяжённых стен
- перемещение объектов
- удаление
- undo / redo
- слои с видимостью и блокировкой
- сохранение и загрузка проекта в JSON
- импорт и экспорт layout DTO из `shared`
- предварительный fog preview для объектов, блокирующих обзор

## Что удобно проверять вручную
1. открыть редактор;
2. выбрать ассет;
3. поставить несколько объектов;
4. нарисовать стену кистью;
5. переместить объект инструментом Move;
6. удалить объект;
7. включить/выключить слой;
8. нажать Undo / Redo;
9. сохранить проект;
10. закрыть и открыть снова;
11. импортировать / экспортировать layout JSON.
